FTW: For The Web!, Gerbil package

This is a gerbil package. For documentation and development on FTW
itself, take a look at https://github.com/drewc/ftw.

Most of the code is gerbil. Gerbil is an opinionated dialect of
Scheme designed for Systems Programming, with a state of the art macro
and module system on top of the Gambit runtime.

See https://github.com/vyzo/gerbil.
